Posibles razones :

  • La tarjeta de crédito introducida quizá no tenga suficientes fondos.
  • El número de tarjeta de crédito o el número CVV no se ha escrito correctamente.
  • Su banco emisor no homologó el CVV o la fecha de vencimiento de su tarjeta de crédito.
  • La dirección de facturación introducida no coincide con la dirección de facturación de su tarjeta de crédito.
  • La tarjeta de crédito presentada ya está en uso. Intente utilizar otra tarjeta.

Requerimientos de software :

Deberá tener una conexión a Internet de banda ancha (DSL o cable).

No podrá jugar juegos, ver películas ni escuchar música si utiliza una conexión de discado.

Para reproducir películas y escuchar música debe descargar e instalar el plug-in de Adobe Flash Player, así como Java de Oracle.

También se recomienda utilizar la versión más reciente de un navegador compatible con HTML5 (es decir, Firefox, Chrome, Explorer).

Para ver una película descargada necesitará un reproductor VLC para poder ver el contenido.

Para jugar, primero deberá descargar Yummy Player, disponible en el área de inicio de sesión de los miembros.

Si aún tiene dificultades técnicas, comuníquese con nuestro equipo de atención al cliente para obtener ayuda.
